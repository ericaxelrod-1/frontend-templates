# Login Page Redesign

## Objective

Redesign the login page utilizing a modern **glassmorphism** aesthetic to make the application feel more premium and effectively utilize screen real estate on desktop devices. The redesign will adhere to UI best practices by ensuring primary login elements (email, password, sign in button, forgot password, create account) are visible above the fold and uniformly styled. The CAPTCHA will be placed in its own distinctly separated container within the new layout, and a scrollbar rendering issue will be resolved.

## End State

- The login page features a vibrant background with a frosted-glass card effect.
- The login form elements (email, password, submit, help links) are grouped cleanly together above the fold on all standard screen sizes.
- The CAPTCHA is housed in a distinct, separate container.
- On desktop, the card utilizes a wider, side-by-side flex/grid layout to minimize dead space.
- A functional scrollbar gracefully handles viewports that are shorter than the login card content.
- Styling is uniform across inputs, buttons, and containers.
